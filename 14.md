# 繰り返し(for文)～沢山のものを動かす（２）～

## 基本形

```cs
for)(初期化（省略可能）;継続条件;更新処理){

//（）内は書かなければtrueの無限ループ
//自力で判定、更新を行い、break;などで抜ける。

  処理A
}

```

## 処理順番
1.初期化
2.継続条件判定
3. (true)処理A
4.更新処理
5.３．から繰り返す
6.(false)ループから抜ける
