# プロパティ(Property)～性質、属性～

## button1のプロパティを観察する

- BackColor
  -背景の色
- Enabled
  -押せなくなる（機能の有無on off）
- Font
  -書式設定、サイズ
- ForeColor
  -文字の色
- Location
  -座標x,y
- Size
  -大きさ
- Text
  -表示文字
- TextAlign
  -ボタン内の文字表示の場所
- Visible
  -ボタンの表示、非表示
- BackgroundImage
  -ボタン背景を画像にする。
- FlatStyle
  -ボタンのスタイル
- Image
  -画像にする
- ImageAlign
  -画像のどこを使うか
  
他にもあるよー
